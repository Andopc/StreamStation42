Seed Local Content

Enter the path to a local AV1 (or other video) file. The system will generate a .torrent file and start seeding so peers can download it. Copy the magnet link and add it to your torrent_channel.json configuration.

Download from Magnet Link

Paste a magnet link from another channel host to download and seed their content. The file will be saved to the specified directory and automatically added to the seeding pool so you help distribute it to other peers.

Note: Requires libtorrent to be installed (pip install python-libtorrent).

How to Create an Indie P2P Channel

  1. Convert your show to AV1 format (recommended for small file sizes):
    ffmpeg -i input.mp4 -c:v libaom-av1 -crf 35 -b:v 0 output.av1
  2. Place the file in a directory like catalog/mychannel/
  3. Use Seed Local Content above to generate a magnet link
  4. Copy confs/examples/torrent_channel.json to confs/mychannel.json and fill in your torrent_streams entries with the magnet links
  5. Restart FieldStation42 — your channel will appear in the guide
  6. Share the magnet links and your .json config with friends so they can join the network and help seed your content!
